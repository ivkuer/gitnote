Git: 分布式版本控制工具
reset （清屏）
Ctrl + L （最后一行移至最上方）
git init (初始化git)
git status （查看状态）
git add （提交到暂存区）
git rm --cached 文件名 （从暂存区删除此文件）
git commit -m "日志信息" 文件名 （提交到本地库，会生成历史版本）
git reflog （显示可引用的历史版本记录）
git log (查看详细版本信息)
git reset --hard 版本号 (版本穿梭)
git branch -v (查看当前分支)
git branch 分支名 (创建一个分支)
git checkout 分支名 (切换分支)
git merge 分支名 (合并此分支)
git remote -v (查看远程库别名)
git remote add 别名 远程库地址 (创建远程库别名)
git remote remove 别名 (删除别名)
git push 别名/ori 要推的分支 (推送到远程库， 没用别名用网址也可以)
git pull 网站/别名 分支  (拉取分支)
git clone 网站 (克隆会自动初始化，创建别名origin，拉取代码)
ssh-keygen -t rsa -C 自己随便设 (设置ssh公钥，用于免密登陆)
